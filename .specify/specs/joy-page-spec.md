# 晚晴小程序 - 乐享页面规格说明

---

## 功能概述

乐享页面是晚晴养老服务小程序的活动与旅游中心，为中老年用户提供丰富多彩的旅游线路、兴趣活动、社交聚会和线上活动，帮助用户享受健康、快乐的晚年生活。

**核心目标**：让用户方便地浏览和参与各类活动，发现生活乐趣。

---

## 页面结构

### 1. 顶部头图区
- **内容**：
  - 大标题："乐享生活"
  - 副标题："发现更多精彩活动"
- **样式**：
  - 绿色渐变背景（#52c41a → #73d13d）
  - 白色文字
  - 标题字号：48rpx（24px）
  - 副标题字号：28rpx（14px）
  - 高度：180px

### 2. 旅游线路推荐区
- **标题**："精选旅游"（带装饰）
- **内容**：3 条旅游线路卡片
- **卡片信息**：
  - 封面图（16:9）
  - 线路名称（如"海南三亚 5 日游"）
  - 价格（如"¥2999 起"）
  - 标签（如"适合老年人"、"慢节奏"）
  - 时长（如"5 天 4 晚"）
- **样式**：
  - 大图在上，信息在下
  - 卡片圆角：16rpx
  - 卡片间距：24rpx
  - 标签：绿色背景，白色文字

### 3. 兴趣活动区
- **标题**："兴趣活动"（带装饰）
- **内容**：3 个兴趣活动卡片
- **卡片信息**：
  - 活动封面图
  - 活动名称（如"太极拳晨练"、"书法艺术交流"）
  - 活动时间（如"每日 7:00"）
  - 活动地点（如"公园广场"）
  - 活动描述（一句话介绍）
- **标签类型**：
  - 文艺：书法、绘画、舞蹈
  - 运动：太极、瑜伽、广场舞
  - 学习：健康讲座、技能培训

### 4. 社交聚会区
- **标题**："社交聚会"（带装饰）
- **内容**：3 个聚会活动卡片
- **卡片信息**：
  - 聚会封面图
  - 聚会名称（如"周末茶话会"、"生日派对"）
  - 时间（如"每周六 14:00"）
  - 地点（如"活动中心"）
  - 参与人数（如"限 20 人"）

### 5. 线上活动区
- **标题**："线上活动"（带装饰）
- **内容**：3 个线上活动卡片
- **卡片信息**：
  - 活动封面图
  - 活动名称（如"健康养生讲座"、"书法直播课"）
  - 时间（如"周三 19:00"）
  - 平台（如"微信直播"、"腾讯会议"）
  - 参与方式（如"扫码加入"）

## 展示内容

### 数据结构

```typescript
// 旅游线路
interface TravelRoute {
  id: string
  name: string
  image: string
  price: number
  duration: string
  tags: string[]
  description: string
}

// 兴趣活动
interface Activity {
  id: string
  title: string
  image: string
  time: string
  location: string
  category: 'art' | 'sports' | 'learning'
  description: string
}

// 社交聚会
interface SocialEvent {
  id: string
  title: string
  image: string
  time: string
  location: string
  maxParticipants: number
  description: string
}

// 线上活动
interface OnlineEvent {
  id: string
  title: string
  image: string
  time: string
  platform: string
  joinMethod: string
  description: string
}
```

### 数据条数
- 旅游线路：3 条
- 兴趣活动：3 条
- 社交聚会：3 条
- 线上活动：3 条

## 交互设计

### 导航跳转
- **点击卡片**：跳转到对应详情页
  - 旅游线路 → `/pages/joy/travel-detail/index?id={id}`
  - 兴趣活动 → `/pages/joy/activity-detail/index?id={id}`
  - 社交聚会 → `/pages/joy/social-detail/index?id={id}`
  - 线上活动 → `/pages/joy/online-detail/index?id={id}`

### 视觉反馈
- **点击效果**：卡片按下时有缩放效果（scale: 0.98）
- **加载状态**：图片加载时显示占位色块
- **空状态**：无数据时显示 "暂无活动" 提示

## 中老年用户适配

### 字体适配
- 所有文字 ≥ 28rpx（14px）
- 标题使用大字号（36rpx+）
- 价格等重要信息使用粗体

### 交互适配
- 卡片足够大，点击区域宽松
- 点击有明确的视觉反馈
- 避免复杂的手势操作

### 视觉适配
- 高对比度配色
- 图片清晰度高
- 避免复杂的动画效果
