# 晚晴社区门店 - 业务架构说明 v2.0

**创建日期**: 2026-01-15
**项目定位**: 社区门店的数字化服务平台
**目标用户**: 社区内及周边的老年人群体

---

## 一、项目定位

### 1.1 核心价值
为社区门店提供数字化服务能力，让老年人通过小程序即可：
- 浏览和购买门店的商品/服务
- 参与社区活动和旅游
- 管理个人健康数据
- 享受便捷的上门服务

### 1.2 商业模式
- **B2C模式**：门店直接服务老年用户
- **O2O闭环**：线上下单 + 线下服务/自提
- **会员储值**：预付费模式，提升用户粘性
- **服务抽佣**：旅游、康养院推荐获取分成

### 2.2 乐享页面（Joy）- 消费服务中心

#### 功能定位
商品购买 + 服务预订 + 活动参与

#### 核心模块（5大板块）

##### 1️⃣ 旅游服务
```
内容: 周边游、国内游、康养旅游
功能: 浏览线路、查看详情、在线报名
盈利: 门店抽佣 10-15%
数据: 线路名称、价格、行程、剩余名额
```

##### 2️⃣ 康养院入住
```
内容: 合作康养院推荐
功能: 浏览康养院、查看环境、预约参观
盈利: 推荐佣金 ¥500-2000/单
数据: 康养院名称、地址、价格、服务特色
```

##### 3️⃣ 社区活动
```
内容: 门店组织的线下活动
功能: 活动列表、在线报名、签到打卡
盈利: 活动报名费、免费活动引流
数据: 活动时间、地点、名额、报名状态
```

##### 4️⃣ 老年商品购买
```
内容: 养生食品、医疗器械、日用品
功能: 商品浏览、加购物车、在线支付
盈利: 商品销售利润 30-50%
数据: 商品图片、价格、库存、规格
```

##### 5️⃣ 上门服务
```
内容: 打扫卫生、维修、陪诊、理发
功能: 服务预约、时间选择、在线支付
盈利: 服务抽佣 15-20%
数据: 服务类型、价格、可预约时间、服务人员
```

### 2.3 颐养页面（Care）- 健康管理中心

#### 功能定位
健康数据管理 + 养生服务

#### 核心模块（5大板块）

##### 1️⃣ 健康档案
```
内容: 个人健康数据记录
功能:
  - 基础信息: 姓名、年龄、血型、过敏史
  - 健康指标: 血压、血糖、心率、体重
  - 就医记录: 就诊时间、诊断结果、处方
  - 数据趋势: 图表展示健康指标变化
数据来源: 手动录入 + 设备同步 + 门店录入
```

##### 2️⃣ 关联设备
```
内容: 智能健康设备绑定
功能:
  - 设备绑定: 扫码绑定手环、血压计、血糖仪
  - 数据同步: 自动同步健康数据到档案
  - 设备管理: 查看设备状态、电量、连接状态
  - 数据提醒: 异常数据自动提醒
支持设备:
  - 智能手环（步数、心率）
  - 电子血压计（血压、心率）
  - 血糖仪（血糖值）
  - 智能体脂秤（体重、体脂）
```

##### 3️⃣ 用药提醒
```
内容: 药品定时服用提醒
功能:
  - 添加药品: 药名、剂量、频次
  - 设置提醒: 每日服药时间
  - 服药记录: 标记已服/未服
  - 家人查看: 子女可查看父母服药记录
提醒方式:
  - 小程序推送通知
  - 短信提醒（可选）
  - 语音电话提醒（VIP服务）
```

##### 4️⃣ 健康咨询
```
内容: 在线健康问诊服务
功能:
  - 图文咨询: 发送文字、图片咨询医生
  - 电话咨询: 预约电话咨询时间
  - 视频咨询: 与医生视频通话
  - 历史记录: 查看咨询历史
服务人员:
  - 门店健康顾问
  - 合作医生/护士
收费模式: 按次收费 or 会员免费
```

##### 5️⃣ 养生内容
```
内容: 养生知识科普
形式:
  - 养生文章: 图文内容，每日推送
  - 养生视频: 短视频形式，1-3分钟
  - 直播课堂: 健康讲座直播
分类:
  - 饮食养生: 食疗、营养搭配
  - 运动健身: 太极、瑜伽、广场舞
  - 疾病预防: 慢性病管理、康复指导
  - 中医养生: 季节养生、穴位按摩
互动功能:
  - 收藏、点赞、分享
  - 评论互动
  - 相关推荐
```


## 三、业务流程设计

### 3.1 商品购买流程
```
浏览商品 → 查看详情 → 加入购物车 → 结算 → 选择收货地址 → 支付(余额/微信) → 订单确认 → 门店配送/自提
```

### 3.2 服务预约流程
```
浏览服务 → 选择服务 → 选择时间 → 填写地址/需求 → 支付定金/全款 → 门店确认 → 服务人员上门 → 完成评价
```

### 3.3 活动报名流程
```
浏览活动 → 查看详情 → 立即报名 → 填写信息 → 支付费用(免费活动跳过) → 报名成功 → 活动提醒 → 签到参与
```

### 3.4 旅游预订流程
```
浏览线路 → 查看详情 → 咨询客服 → 确认出行 → 支付定金 → 签订合同 → 支付尾款 → 出行游玩 → 游后评价
```

### 3.5 康养院预订流程
```
浏览康养院 → 查看详情 → 预约参观 → 实地考察 → 确认入住 → 支付费用 → 办理入住
```

## 四、会员体系设计

### 4.1 会员等级

| 等级 | 升级条件 | 权益 |
|------|----------|------|
| **普通会员** | 注册即可 | 基础功能 |
| **VIP会员** | 充值¥500或累计消费¥2000 | 9.5折、专属客服、免费健康咨询 |
| **尊享会员** | 充值¥2000或累计消费¥10000 | 9折、优先服务、家庭账户(3人)、上门服务费减半 |

### 4.2 储值优惠
```
充值¥500  → 到账¥520   (送¥20)
充值¥1000 → 到账¥1050  (送¥50)
充值¥2000 → 到账¥2100  (送¥100)
充值¥5000 → 到账¥5300  (送¥300)
```

### 4.3 积分体系
- **消费积分**: 消费¥1 = 1积分
- **活动积分**: 参与活动、签到获得积分
- **积分兑换**: 100积分 = ¥1（可抵扣现金）
- **积分商城**: 兑换礼品、优惠券

## 五、盈利模式总结

### 5.1 收入来源

| 来源 | 模式 | 预期占比 |
|------|------|----------|
| **商品销售** | 老年用品销售差价 | 35% |
| **服务抽佣** | 上门服务、旅游抽佣 | 25% |
| **会员储值** | 预付费资金沉淀 | 20% |
| **活动收费** | 活动/课程报名费 | 10% |
| **康养推荐** | 康养院推荐佣金 | 10% |

### 5.2 单用户经济模型
```
假设单老年用户:
- 月均消费: ¥500 (商品+服务)
- 年均消费: ¥6000
- 毛利率: 30%
- 年毛利: ¥1800/用户

1000用户 → 年毛利¥180万
5000用户 → 年毛利¥900万
```
